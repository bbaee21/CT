# SWEA 1976

> 시각 덧셈
>
> 시 분으로 이루어진 시각을 2개 입력 받아, 더한 값을 시 분으로 출력하는 프로그램을 작성하라.
>
> (시각은 12시간제로 표시한다. 즉, 시가 가질 수 있는 값은 1시부터 12시이다.)

```python
T = int(input())
 
for tc in range(1, T+1):
    h1, m1, h2, m2 = map(int, input().split())
 
    h = h1 + h2
    m = m1 + m2
 
    carry = m // 60      # 몫 올림수
    m = m % 60          # 분에 대한 나머지
    h += carry
 
    if h % 12 == 0 and h > 0:
        h = 12
    else:
        h %= 12
 
    print("#{} {} {}".format(tc, h, m))
```
