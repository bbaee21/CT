# SW Expert 1943

> 입력으로 1개의 정수 N 이 주어진다.
>
> 정수 N 의 약수를 오름차순으로 출력하는 프로그램을 작성하라.

1. 약수니까, 나누기로 접근해야겠다.
2. 나누기로 접근하는데, 나머지가 0인 숫자들을 찾으면 되겠다.
3. 그리고 숫자 모르고 정렬해야겠다.

```python
N = int(input())
lst = ''
for num in range(1, 1001):
    if N % num == 0:
        lst += str(num)
        lst += ' '
print(lst)
```
